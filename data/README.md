# Data provenance

Initial experiments generate synthetic data locally under `data/synthetic/`; generated CSVs are ignored by Git.

Public datasets are not bundled. A future adapter must document the source URL, license, version, transformation steps, and redistribution terms in `data/public/README.md` before any file is used. Never add credentials, private measurements, personal data, university/student/WRepo data, employer data, OPMobility data, Mitsubishi Fuso data, or unpublished research material.
